# 🎣 21 Trucos Secretos de Anzuelo — Landing Page

Página de vendas otimizada para conversão de um ebook digital de pesca.

## 📋 Sobre o Projeto

Landing page em espanhol para vender o ebook **"21 Trucos Secretos de Anzuelo que Multiplican Tus Capturas"** por $10 USD.

- **Produto:** Ebook digital (PDF)
- **Público:** Homens que gostam de pesca recreativa
- **Tráfego:** YouTube e redes sociais
- **Preço:** $10 USD (pagamento único)

## ⚙️ Como usar

1. Clone o repositório:
```bash
git clone https://github.com/Yury98Garcia/landingpageesp1.git
```

2. Abra o arquivo `index.html` no navegador.

3. **Antes de publicar**, substitua todos os links `https://tu-link-de-checkout.com` pelo seu link real de checkout (Hotmart, Kiwify, Stripe, Gumroad, etc.).

4. Crie os arquivos `terminos.html` e `privacidad.html` com seus termos legais.
